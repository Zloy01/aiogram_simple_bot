<h1 align="center">Simple Bot AIOgram 3.x</h1>

> [!CAUTION]
> Make sure to change the `.env.example` file to `.env` and the data in it

- Clone a repo
```bash
git clone https://github.com/Zloy01/aiogram_simple_bot.git
```
- Change dir
```bash
cd aiogram_simple_bot
```
- Run
```bash
python main.py
```
